# After Platform Visibility: Public Derived Materials

This repository is a placeholder for public-safe derived materials associated
with the manuscript:

**After Platform Visibility: Heat, Structural Review, and Gender Controversies
on Weibo**

## Status

Data files are coming soon.

## Planned scope

The repository is intended to host de-identified, derived, aggregate, or
redacted materials only. It will not include raw Weibo post text, direct post
identifiers, user identifiers, URLs, user handles, profile fields, exact case
dates, event names, top terms, short-context fields, annotator notes, raw
annotation packets, credential-bearing scripts, request headers, cookies, API
keys, or other sensitive operational materials.

## Citation

Please cite the manuscript after publication. Citation details will be added
when available.

## Review Anonymity

This repository is prepared for anonymous peer review. Author-identifying
contact information will be added after peer review or publication when
appropriate.
